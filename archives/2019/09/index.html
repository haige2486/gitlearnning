关于远程部署到github上的说明

  1. 创建github的主页，即建立repo名为：[用户名].github.io

  2. cnpm install --save hexo-deployer-git

  3. 修改目录下的_config.yml配置文件

    deploy:
    type: git
    repo: 你的git主页地址
    branch: master #设置本地为master，git上的代码每次会以本地为准，可不设

  4. git config —global user.name “nameVal”

  5. git config —global user.email “eamil@qq.com”,步骤4、5为git全局设置，用于部署连接
